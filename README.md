  </p>

  <p>
    <b>Role</b><br/>
    Backend Developer
  </p>

  <p>
    <b>Tech Stack</b><br/>
    <img src="https://img.shields.io/badge/Java-007396?style=for-the-badge&logo=Java&logoColor=white"/>
    <img src="https://img.shields.io/badge/Spring-6DB33F?style=for-the-badge&logo=Spring&logoColor=white"/>

    <img src="https://img.shields.io/badge/Nginx-009639?style=for-the-badge&logo=nginx&logoColor=white"/>
  </p>

  <p><b>What I Focused On</b></p>
  <ul>
    <li>
      Spring 기반 <b>MVC 아키텍처</b>를 학습 목표로 설정하고  
      Controller–Service–Repository 계층 분리를 지키며 백엔드 구조 설계
    </li>
    <li>
      <b>API 설계 및 명세서 작성</b>을 직접 담당하여  
      프론트엔드와의 협업 기준 수립
    </li>
    <li>
      User / Question / Comment 도메인 중심으로  
      CRUD 및 비즈니스 로직 구현
    </li>
    <li>
      <b>AWS EC2</b> 기반 Ubuntu 서버를 직접 생성하고  
      키 페어 발급 및 보안 그룹 설정을 통한 접근 제어 구성
    </li>
    <li>
      SSH 접속 후 Linux 명령어를 활용해  
      서버 환경 설정 및 애플리케이션 실행
    </li>
    <li>
      <b>Nginx</b>를 활용한 웹 서버 설정으로  
      백엔드 서비스 외부 접근 가능하도록 구성
